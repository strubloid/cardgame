type: Fire
life: 5

attack: 1
mana_cost: 3
resonance: false
mana_generation_bonus_if_same_type: 0
tags: Support, Shield

rules:
  - On play: Give a friendly card +2 Shield.
  - If that card is Fire, it also gains +1 Attack this turn.

synergy:
  - Enables Blazebound Guard triggers; buffs Inferno Drake swings.

status_reference:
  Burned: target takes 1 damage at start of its owner's turn
  Soaked: target has -1 Shield (min 0) while Soaked
  Rooted: target can't gain Shield while Rooted
  Aerial: attacker ignores 1 Shield when dealing damage

element_fusions:
  Steam: Burned + Soaked => next hit deals +1 damage
  Mud: Soaked + Rooted => target can't attack next turn
  Wildfire: Burned + Aerial => Burn spreads to another enemy if any
  Sandstorm: Rooted + Aerial => target Attack -1 next turn
